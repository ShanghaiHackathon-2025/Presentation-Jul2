# 问题
### 中国中学生心理健康问题的日益严峻

<br/>

- **普遍心理健康问题：**
  - **焦虑、抑郁高发**：约 **25%** 的中学生有轻度至重度抑郁症状，焦虑与抑郁是主要问题。
  - **高压力**：12.9%至26.6%的学生报告心理压力过大。

- **自伤与自杀意念：**
  - **自杀率上升**：青少年自杀率持续上升。
  - **自杀意念普遍**：8.4%有自杀意念，3.8%有自杀计划。

- **影响广泛：**
  - **学业与人际关系受损**：压力导致睡眠不足、情绪问题和竞争性敌意，影响身心与社会融合。

# 拟议研究设计
### 结合定量问卷和定性访谈/焦点小组的混合方法

**理由：**
- **定量问卷：** 提供广度，捕捉患病率、态度和偏好模式（“是什么”、“有多普遍”）。
- **定性访谈/焦点小组：** 提供深度，深入了解情感因素、个人经历和细微视角（“为什么”）。
- **优势互补：** 兼顾可推广数据和深层动机理解，尤其适用于敏感话题。

# 方法论框架
### 定量部分（问卷）

**设计原则：** 明确目标、简洁明了、封闭式问题为主、评分量表（李克特）、“不适用”选项、问题顺序、包容性语言、引言和时间估算、预测试、必要时强制性问题、随机抽样。
- **内容领域：**
	- 感知压力水平 (学业、同伴、家庭、生活方式)
	- 当前应对机制 (求助、爱好、数字设备、回避)
	- 对心理健康的态度 (耻辱感、舒适度、有效性、信任)
	- 偏好的支持类型 (校内、外部、同伴、家长教育、在线)
	- 寻求帮助的障碍 (成本、认知、父母、保密、等待时间)
	- 人口统计信息 (年龄、性别、年级、城乡、独生子女、父母教育/收入)

# 拟议研究设计

### 焦点小组方法

**设计原则：** 明确目标、简洁明了、封闭式问题为主、李克特评分量表、不适用选项、合理问题顺序、包容性语言、引言与时间估算、预先测试、必要时强制性问题、随机抽样。

**优势：** 丰富见解、降低偏见、深入探究、安全环境、探索社会动态、高效时间利用。

**劣势：** 耗时、群体思维/主导风险、复杂日程安排、主观性强、样本量小。

**额外设计原则：** 明确目的和范围、灵活协议、开放式问题、建立融洽关系、目标抽样、主持人培训、安全舒适环境。

## 其他

### 与未成年人研究的伦理考量

**伦理审查：** 研究必须遵循伦理标准，确保参与者权益和福祉。
- **强制性IRB批准：** 必须获得机构审查委员会或同等伦理委员会批准。
- **知情同意和同意：**
	- 自愿参与、随时退出。
	- 适合年龄的信息呈现。
	- 未成年人需获得“同意”。
	- 父母/法定监护人明确许可。
- **保密性和隐私：** 严格保护个人信息，处理敏感信息（如自伤/自杀意念）需有明确报告协议。
- **最小化潜在伤害和管理权力不平衡：** 确保参与不会对其护理、学业或关系产生负面影响。
- **适当的激励：** 激励不构成不正当影响或胁迫。
- **中国背景下的挑战：** 父母权威、“面子”观念、心理疾病耻辱感，需异常敏感和策略性。

# 关键要点
### 关键要点

**心理健康问题普遍：** 中国中学生面临严重的心理健康挑战，焦虑、抑郁和自杀意念普遍存在。

* **复杂问题：** 中国青少年心理健康是复杂且多方面的问题，受多重因素深刻影响。
* **研究基石：** 深入的混合方法需求研究不可或缺，提供具体可操作见解，确保干预措施文化敏感、可接受、有效。
* **协同方法：** 促进学生心理健康的真正支持生态系统，需要政府、学校、家庭、社区协同和协调一致。
* **战略转向：** 从被动危机管理转向积极、整体的福祉框架，系统性解决耻辱感，提高可及性，赋予学生韧性。
